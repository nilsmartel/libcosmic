pub type LineRef<'s> = LineSlice<&'s [u8]>;
Expand description

An reference to a single line of pixels. May go across the whole image or just a tile section of it.

This line contains an immutable slice that all samples will be read from.

Where this line is located inside the image.

The raw bytes of the pixel line, either &[u8] or &mut [u8]. Must be re-interpreted as slice of f16, f32, or u32, according to the channel data type.
